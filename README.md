# Smart Expense Tracker

A modern, high-end fintech application designed to track expenses intelligently. Inspired by the sleek aesthetics of Stripe and Linear, this dashboard offers seamless interaction, dynamic visualizations, and AI-driven insights to help users manage their personal finances effectively.

## 🚀 Key Features

*   **Premium UI/UX:** A minimal, dark-themed, glassmorphism-inspired aesthetic that looks and feels like a top-tier financial product.
*   **AI-Driven Financial Insights:** Integrated with Google Gemini 2.5 Flash to automatically analyze your spending habits and provide smart, actionable financial advice.
*   **Interactive Data Visualization:** 
    *   Interactive Donut and Bar charts representing your expenses.
    *   Click-to-filter functionality to drill down into specific expense categories instantly.
*   **Smart Validation Logic:** Context-aware validation ensures that your expense notes and descriptions perfectly align with the selected financial categories.
*   **Advanced Data Controls:** Seamless search and comprehensive sorting capabilities across all expense items.
*   **Responsive Architecture:** Works beautifully across different screen widths with smooth micro-animations.

## 🛠️ Technology Stack

*   **Frontend:** HTML5, modern vanilla CSS (Variables, Flexbox/Grid, Animations), and modular JavaScript.
*   **AI Engine:** Google Gemini SDK integration via `ai.js`.
*   **Data Visualization:** Chart.js integration for beautiful, animated financial graphics.
*   **Design:** Custom particles and effects implemented for a dynamic experience.

## ⚙️ Getting Started

To run this project locally, you don't need any complex build steps. Just serve the files locally!

1.  Clone this repository:
    ```bash
    git clone https://github.com/adi-jha14/Smart-Expense-Tracker.git
    ```
2.  Navigate to the directory:
    ```bash
    cd Smart-Expense-Tracker
    ```
3.  Open `index.html` in your browser or run via a local development server like VSCode Live Server.
4.  **Configuring the AI Assistant:**
    To utilize the AI Insights generation powered by Gemini:
    *   Get a free API key from [Google AI Studio](https://aistudio.google.com/).
    *   Open `js/ai.js` in your code editor.
    *   Locate `var API_KEY = '...';` near the top of the file (Line 6).
    *   Replace the default key with your new Gemini API key.
